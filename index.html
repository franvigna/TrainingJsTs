Introducción

La idea de este material es matchear conceptos que ya son conocidos con los nuevos. La mejor forma que hay de aprender es haciendo, pero antes de hacer tienen que conocer los conceptos básicos de la tecnología con la que nos manejamos. Con esta guía, quiero dar un enfoque claro y concreto de a qué va la cuestión.

Glosario

  • Web Part: Componentes independientes y configurables dentro de páginas en SharePoint.
  • SPFx: Framework de desarrollo para personalizar SharePoint Online mediante Web Parts.
  • App Catalog: Repositorio en SharePoint para soluciones personalizadas.
  • Workbench: Entorno local para probar Web Parts antes del despliegue.
  • SharePoint REST API: Interfaz HTTP para interactuar con SharePoint.
  • Task runner: Herramienta que automatiza tareas repetitivas como Gulp.
  • UI (User Interface): Interfaz visual para el usuario.

Herramientas en SPFx

  • Node.js: Entorno para ejecutar JavaScript fuera del navegador.
  • NVM: Gestor de versiones de Node.js.
  • npm: Gestor de paquetes para Node.js.
  • Yeoman: Generador de plantillas.
  • Gulp: Automatizador de tareas.
  • TypeScript: Superset de JavaScript con tipado.
  • Webpack: Empaquetador de módulos.
  • React: Biblioteca para interfaces de usuario.
  • Fluent UI: Componentes UI estilo Microsoft.
  • Workbench: Entorno local para testeo.
  • App Catalog: Publicación de Web Parts.
  • PnPjs: Librería para consumir servicios de SharePoint.
