¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se ejecuta en el navegador y permite crear experiencias interactivas en sitios web.

Variables

En JavaScript se pueden declarar variables con var, let y const. Hoy en día, se recomienda usar let y const.

// Ejemplo
let nombre = "Juan";
const PI = 3.14;

Tipos de datos

  • String → Texto
  • Number → Números (enteros o decimales)
  • Boolean → true / false
  • Undefined → Valor no definido
  • Null → Valor nulo
  • Object → Colección de datos
