2. Objetos y Arrays en JavaScript

← Volver al índice

Objetos en JavaScript

Los objetos son colecciones de propiedades, donde cada propiedad se compone de una clave (nombre) y un valor. 


const persona = {
  nombre: "Ana",
  edad: 30,
  saludar: function() {
    console.log("Hola, soy Ana");
  }
};

Acceso a propiedades

Puedes acceder a las propiedades de un objeto utilizando la notación de punto o la notación de corchetes. 


console.log(persona.nombre); // Notación de punto
console.log(persona["edad"]); // Notación de corchetes

Arrays en JavaScript

Los arrays son listas ordenadas de elementos, que pueden ser de cualquier tipo de dato. 


const frutas = [
  "manzana", 
  "banana", 
  "naranja"
];

Acceso a elementos de un array

Puedes acceder a los elementos de un array utilizando su índice, comenzando desde 0. 


console.log(frutas[0]); // "manzana"

Métodos comunes de arrays

Algunos métodos útiles para manipular arrays son:

  • push(): Agrega un elemento al final del array.
  • pop(): Elimina el último elemento del array.
  • shift(): Elimina el primer elemento del array.
  • unshift(): Agrega un elemento al inicio del array.
  • map(): Crea un nuevo array con los resultados de la llamada a una función para cada elemento.
  • filter(): Crea un nuevo array con todos los elementos que cumplan la condición implementada por la función dada.
