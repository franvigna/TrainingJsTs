¿Qué es TypeScript?

TypeScript es un lenguaje de programación desarrollado por Microsoft que se basa en JavaScript, pero le agrega tipado estático y otras funcionalidades orientadas a grandes aplicaciones.

Sintaxis básica

La sintaxis de TypeScript se basa en la de JavaScript, pero incorpora reglas adicionales para declarar tipos de datos y estructuras más robustas. Incluye normas sobre cómo definir variables con tipos específicos, cómo estructurar funciones con parámetros tipados, y cómo crear interfaces, clases y tipos personalizados para garantizar mayor claridad y seguridad en el código. Estas reglas permiten que los errores se detecten antes de ejecutar el programa y ayudan a que el código sea más legible, mantenible y fácil de escalar..

Variables

En TypeScript se pueden declarar variables con var, let y const. Hoy en día, se recomienda usar let y const. 

// Ejemplo
let nombre: String = Juan;
const pi: Number= 3.14;

Tipos de datos primitivos

  • String: Cadenas de texto.
  • Number: Números, ya sean enteros o decimales.
  • Boolean: Valores lógicos, true o false.
  • Null: Representa la ausencia intencional de un valor.
  • Undefined: Indica que una variable no ha sido asignada.
  • Symbol: Valores únicos e inmutables, introducidos en ES6. 
let texto = "Hola";
let numero = 123;
let activo = true;
let vacio = null;
let sinValor;
let simbolo = Symbol("id");

Operadores

Se utilizan para realizar operaciones: aritméticas, de comparación y lógicas. 

// Aritméticos
let suma = 5 + 3;
// Comparación
let esIgual = (10 == "10");
// Lógicos
let resultado = true && false;

Condicionales

let edad = 18;
if (edad >= 18) {
  console.log("Es mayor de edad");
} else {
  console.log("Es menor");
}

let dia = 1;
switch (dia) {
  case 1: console.log("Lunes"); break;
  case 2: console.log("Martes"); break;
  case 3: console.log("Miercoles"); break;
  case 4: console.log("Jueves"); break;
  case 5: console.log("Viernes"); break;
  case 6: console.log("Sabado"); break;
  case 7: console.log("Domingo"); break;
  default: console.log("Otro día");
}

Bucles

// for
for (let i = 0; i < 3; i++) {
  console.log(i);
}

// while
let j = 0;
while (j < 3) {
  console.log(j);
  j++;
}
